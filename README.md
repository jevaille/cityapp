# City App

A comprehensive Flutter application for citizens to access city services and information.

## Features

- **Authentication**: Login, Register, Forgot Password, OTP Verification
- **News**: Latest city news and announcements
- **Events**: City events and activities
- **Job Portal**: Job listings and applications
- **Business Bureau**: Business permit applications and renewals
- **Civil Registry**: Birth, Marriage, and Death certificates
- **Citizen Reports**: Report issues and track status
- **Emergency Services**: Hotlines, evacuation centers, SOS alert
- **City Directory**: Government offices and officials
- **Maps**: Interactive maps with locations
- **Notifications**: Push notifications and alerts
- **Profile**: User profile management and settings
